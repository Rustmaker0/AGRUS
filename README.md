
# AGRUS (лабораторный прототип)

Локальная full‑stack версия с разделением ролей (клиент / мастер) и JSON‑базой.

## Запуск
1. Установите Node.js 18+.
2. В папке проекта выполните:
   ```bash
   npm i
   npm start
   ```
3. Откройте в браузере http://localhost:3000

## Готовые аккаунты
- Мастер: **master@agrus.test** / **1234**
- Клиент: **client@agrus.test** / **1234**

## Структура
- `server.js` — API + статика из `public/`
- `db.json` — "база данных" (JSON)
- `public/` — фронтенд-страницы и скрипты

## Замечания
- Авторизация токеном: после входа токен хранится в LocalStorage и передаётся как Bearer.
- Это учебный код, без претензии на продакшн‑безопасность.
